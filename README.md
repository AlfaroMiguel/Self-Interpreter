Como compilar el proyecto:
Hacer "cmake ./src"
y luego dirigirse a la carpeta que se quiere compilar (client/server) y hacer "make"

Librerias necesarias:
gtkmm-3.0
goocanvas-2.0
bison y flex (si se quiere generar el parser)
